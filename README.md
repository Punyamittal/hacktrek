# AI Automate 2026 - Hackathon Website

A modern, high-performance hackathon website built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Sections Implemented

- Hero / landing with animated futuristic background
- About section
- Themes cards grid with hover effects
- What to build section
- Submission requirements checklist UI
- Prizes with trophy highlight and grand prize
- Judges card
- Judging criteria with animated progress bars
- Timeline
- Footer with social links and contact

## Run Locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Build for Production

```bash
npm run build
npm run start
```
